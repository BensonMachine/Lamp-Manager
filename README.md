   
# LampMane

***An alternative to the "XAMPP Launcher" for the Ubuntu OS and other Linux distros.***  

##

Users who do not like the "Command Line" will prefere **LampMane** because it is a GUI machine! 

**Apache :**  
`Start` **|** `Stop` **|** `Restart`  

**MySQL :**  
`Start` **|** `Stop` **|** `Restart`
